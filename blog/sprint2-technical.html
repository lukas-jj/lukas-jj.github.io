How to use Class and ID in HTML/CSS

16 November 2020

When using HTML, there will come a point where you want to start styling your page. Perhaps you want all of your titles to be underlined, or a paragraph of text to be blue. In order to know what to style, CSS uses selectors.

Class

Class is used when you want to style multiple elements in the same way. For example, if you have multiple headers in your page which you'd like to be green and underlined, you would give them all the same class name (E.g. class="amazing-headers"). This saves you from writing out the same style multiple times. To select a class, you must put a '.' before your class name in your CSS file. Here is an example:

In HTML: 

<h4 class="amazing-headers">My Amazing Header</h4>
<h4 class="amazing-headers">My Amazing Header Part 2: Revenge of the Headers</h4>
<h4 class="amazing-headers">My Amazing Header Part 3: This Time It's Personal</h4>
In CSS: 
.amazing-headers {color: green; text-decoration: underlined;}

In the DOM (What is displayed):

My Amazing Header

My Amazing Header Part 2: Revenge of the Headers

My Amazing Header Part 3: This Time It's Personal

ID

ID is used when you want to specifically style one element. For example, if I want only one of my headers to be red and not underlined, but the rest green and underlined, I would give my red header an ID selector (E.g. ID="amazing-red-header"). To select an element's ID, you must put a '#' before your ID name in your CSS file. Here is an example:


In HTML: 
<h4 class="amazing-headers">My amazing header</h4>
<h4 class="amazing-headers">My Amazing Header Part 2: Revenge of the Headers</h4>
<h4 id="amazing-red-header">My Amazing Header Part 3: This Time It's Personal</h4>
In CSS: 
#amazing-red-header {color: red; text-decoration: none;}

In the DOM (What is displayed):

My Amazing Header

My Amazing Header Part 2: Revenge of the Headers

My Amazing Header Part 3: This Time It's Personal

Recap

To sum up, if we want to style multiple elements within our HTML using CSS we use class selectors. If we want to style individual elements, then we use ID selectors. Inside CSS, we use a '.' before writing the name of the class, and a # before the name of an ID. A silly mnemonic I used for remembering this was ID uses a '#' because they're individualistic instagram influencers, whereas classes use a '.' because they are a grain of sand which collectively make up a beach. Ridiculous, right?
